# KinectSample

Collection of early Kinect sample experiments.

## Quickstart

1. Open the solution in a Visual Studio version compatible with the project files.
2. Install legacy Kinect SDK/runtime.
3. Connect a supported Kinect sensor and run the desired sample.

## Sample catalog

- Capture sample: demonstrates frame capture flow.
- Gesture sample: baseline gesture-processing experiments.
- Sensor init sample: startup and device detection checks.
